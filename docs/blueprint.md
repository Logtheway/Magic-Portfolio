# **App Name**: Commander Compendium

## Core Features:

- Game Data Entry: Users can log detailed game outcomes, including participating players, chosen commanders, color combinations, and the game winner. This data is used to calculate win rates.
- Group Management: Create and manage multiple distinct player groups to keep game data separate and track different play circles. Users can also designate themselves within each group.
- Personal Stat Dashboard: A dedicated page displaying the logged-in user's individual statistics, including overall win rates, win rates per commander, and win rates by color combination across all games.
- Group Analytics View: View comprehensive win rate analytics for a selected group, showing top-performing players, commanders, and color combinations within that specific group.
- Interactive Life Tracker: A customizable in-game life counter tool, allowing users to set starting life totals, number of players, and visual player orientation for multi-player games.
- AI Strategic Insight Tool: An AI tool that analyzes recorded game outcomes to provide personalized strategic suggestions, highlight commander synergies, or identify common tactical patterns based on the user's data.

## Style Guidelines:

- Color scheme: Dark theme, reflecting the strategic and arcane nature of Magic: The Gathering. Primary color (interactable elements): A rich, deep indigo (#503399), providing a sense of depth and focus. Background color: A very dark, subtle purplish-gray (#231f2e), to enhance content visibility and immerse the user in the data. Accent color (highlights, calls to action): A vibrant yet refined blue (#7093F7), for clear distinction and energetic accents.
- Headline and numeric text font: 'Space Grotesk' (sans-serif), for a modern, slightly technical, and clear presentation of data. Body and explanatory text font: 'Inter' (sans-serif), chosen for its excellent readability and neutral, objective appearance, ideal for displaying detailed game statistics and group information.
- Use stylized icons drawing inspiration from Magic: The Gathering mana symbols and elemental energies where appropriate, alongside clean, minimalist UI icons for navigation and data filtering actions. Ensure all icons are legible and intuitive against the dark theme.
- Employ a dashboard-centric layout for statistics, ensuring clear visual separation between personal and group data. The life tracker will prioritize a spacious, touch-friendly design optimized for in-game use, with configurable player arrangement mirroring real-life table setups. Content will be responsive, adapting smoothly to various screen sizes.
- Implement subtle and purposeful animations, such as smooth transitions for updating statistics, filtering data, or switching between groups. The life counter will feature minor visual feedback animations, like a brief flash or subtle particle effect, when life totals are adjusted, enhancing user interaction.
